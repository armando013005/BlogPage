Getting started

Creating a static website is easy. The first step for getting started is to install the prerequisites.

Windows

This workshop uses winget as a packed manager. winget should be installed in systems with windows 10 and later

MacOS

This workshop uses homebrew, to install homebrew run the next text in your terminal shell
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

installing git

Git a free and open source version control system. We are going to use this as a tool to get our templates from the internet.

windows

using winget run the following command in the terminal. winget install --id Git.Git -e --source winget
