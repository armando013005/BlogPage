## Getting started
Creating a static website is easy. The first step for getting started is to install the prerequisites.

#### `Windows`
This workshop uses [winget](https://learn.microsoft.com/en-us/windows/package-manager/winget/#install-winget) as a packed manager. winget should be installed in systems with windows 10 and later


#### `MacOS`
This workshop uses [homebrew](https://docs.brew.sh/Installation), to install homebrew run the next text in your terminal shell<br> ``/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"``


### installing git 
[Git](https://git-scm.com/) a free and open source version control system. We are going to use this as a tool to get our templates from the internet.

#### windows<br>
using winget run the following command in the terminal.
```winget install --id Git.Git -e --source winget```

- **Windows**
  - Install Git
  - Install Hugo
  - Run `hugo new site mysite`

- **macOS**
  - Install Homebrew
  - Install Hugo
  - Create your project

- **Linux**
  - Install Git
  - Install Hugo
  - Create your project
